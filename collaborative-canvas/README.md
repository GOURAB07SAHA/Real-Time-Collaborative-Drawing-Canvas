# Collaborative Canvas

This is a real-time collaborative drawing application built with Node.js, Express, Socket.io, and vanilla JavaScript.

## Features

*   Real-time drawing with other users
*   Undo/redo functionality
*   Color and stroke width selection
*   Eraser
*   User cursor indicators

## How to run

1.  Install dependencies: `npm install`
2.  Start the server: `npm start`
3.  Open your browser and go to `http://localhost:3000`
